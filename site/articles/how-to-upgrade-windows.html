How to Upgrade to Windows 11 From Windows 10

By The Tech Brief Editorial Team Published: March 1, 2025 ~10 min read

Microsoft ended mainstream support for Windows 10 in October 2025. If you're still running Windows 10 and your PC meets the requirements, upgrading to Windows 11 is free and keeps your files and applications intact. This guide walks you through the entire process — from the compatibility check to the first boot on Windows 11.

Good to know: Upgrading from Windows 10 to Windows 11 preserves your files, apps, and most settings. It is not a clean install. If you want a completely fresh start, you can choose a clean installation after the upgrade using the Reset this PC option.

Step 1: Check If Your PC Is Compatible

Windows 11 has stricter hardware requirements than Windows 10. The key requirements are:

  • Processor: 1 GHz or faster, 2 or more cores, on a compatible 64-bit processor or SoC. Intel 8th generation (Core i3-8xxx / i5-8xxx / i7-8xxx) or later; AMD Ryzen 2000 series or later.
  • RAM: 4 GB minimum
  • Storage: 64 GB minimum free space
  • TPM: Trusted Platform Module (TPM) version 2.0 — this is the requirement that blocks most older PCs
  • Firmware: UEFI, Secure Boot capable
  • Display: HD (720p) or higher, 9 inches diagonal minimum

Run PC Health Check

Microsoft provides a free tool called PC Health Check that instantly tells you whether your PC qualifies. Search for "PC Health Check" in the Microsoft Store or Windows Settings and run the check. If your PC passes, you're ready to proceed. If it fails, the tool tells you exactly which requirement isn't met.

Check TPM status manually

If you're unsure whether your PC has TPM 2.0 enabled, press Windows Key + R, type tpm.msc and press Enter. The TPM Management window will show the manufacturer and specification version. If it shows "Compatible TPM cannot be found," you may need to enable TPM in your BIOS/UEFI settings.

Step 2: Back Up Your Data Before Upgrading

Although the upgrade process is designed to preserve your files, unexpected failures during OS upgrades can cause data loss. A backup takes 15–30 minutes and could save you hours of grief.

Option A: Windows Backup (Built-in)

  1. Open Settings > Update & Security > Backup
  2. Connect an external drive with at least 50 GB free
  3. Click Add a drive and select your external drive
  4. Toggle Automatically back up my files and click More options
  5. Click Back up now and wait until the backup completes

Option B: Create a System Image

For a complete snapshot of your entire drive — including Windows, apps, and files — search for Control Panel, go to System and Security > Backup and Restore (Windows 7), and select Create a system image. This takes longer but lets you roll back to exactly where you were if something goes wrong.

Step 3: Check for the Windows 11 Upgrade in Windows Update

For compatible PCs, Microsoft delivers the Windows 11 upgrade through Windows Update. This is the safest and most reliable method.

  1. Open Settings
  2. Go to Update & Security > Windows Update
  3. Click Check for updates
  4. If your PC is eligible, you should see a banner reading "Windows 11 is ready — and it's free!"
  5. Click Download and install
  6. Accept the software licence agreement
  7. The download will begin (typically 5–8 GB)

If the Windows 11 offer does not appear, your PC may not yet be in the current rollout wave, or it may not meet requirements. Continue checking over the following days or use the Installation Assistant below.

Step 4: Alternative — Use the Windows 11 Installation Assistant

If Windows Update isn't showing the offer yet but your PC is confirmed compatible, you can trigger the upgrade manually using Microsoft's Installation Assistant.

  1. Go to microsoft.com/software-download/windows11
  2. Under "Windows 11 Installation Assistant," click Download Now
  3. Run the downloaded file
  4. Accept the licence terms and click Accept and install
  5. The assistant downloads Windows 11 and prepares the installation
  6. When prompted, click Restart now — do not interrupt the process

What to Expect During the Upgrade

The upgrade consists of several phases that happen automatically. Your PC will restart multiple times — this is normal. The process typically takes 30–90 minutes depending on your internet speed and hardware. Keep the computer plugged in to power and do not force a shutdown during any phase.

When the process is complete, you'll arrive at the Windows 11 desktop with your files, applications, and settings intact. A brief setup wizard may appear asking about privacy settings and Microsoft account preferences.

What If My PC Isn't Compatible?

If your hardware doesn't meet Windows 11 requirements, you have a few options. You can continue using Windows 10 until you're ready to buy a new PC — Microsoft is offering Extended Security Updates (paid) for Windows 10 through October 2028 for consumers. Alternatively, some users have installed Windows 11 on unsupported hardware using the registry bypass method, but this means you may not receive future updates and Microsoft does not support this configuration. For most users, the practical answer is to budget for a new PC that ships with Windows 11 pre-installed.
